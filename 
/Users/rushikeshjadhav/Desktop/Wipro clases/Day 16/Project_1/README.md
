# Customer Engagement Portal

## Project Overview
A responsive customer engagement portal built using HTML5, CSS3, JavaScript, and Bootstrap 5. This project demonstrates modern frontend development practices including semantic markup, responsive design, form validation, and asynchronous data handling.

## Run Instructions

### Method 1: Direct Browser Opening
1. Navigate to the project folder: `Day 16/Project_1/`
2. Open `index.html` in any modern web browser (Chrome, Edge, Firefox, Safari)

### Method 2: Local Server (Recommended)
```bash
# Using Python 3
cd Day\ 16/Project_1/
python3 -m http.server 8000

# Then open http://localhost:8000 in your browser
```

### Method 3: VS Code Live Server
1. Open the project folder in VS Code
2. Install "Live Server" extension
3. Right-click on `index.html` and select "Open with Live Server"
