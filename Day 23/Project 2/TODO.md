# Day 23 - Project 2: SPA Templates Setup

## Objective
Create 8 different SPA templates to demonstrate various approaches to building Single Page Applications.

## Templates to Create

### 1. Vanilla SPA (Manual with Parcel)
- Directory: `1-vanilla-spa-parcel/`
- Setup: npm init, install parcel, create src/ structure
- Run: `npx parcel src/index.html`

### 2. Vite Vanilla
- Directory: `2-vite-vanilla/`
- Setup: `npm create vite@latest` (Vanilla template)
- Run: `npm run dev`

### 3. Vite Vanilla + TypeScript
- Directory: `3-vite-vanilla-ts/`
- Setup: `npm create vite@latest` (Vanilla + TypeScript template)
- Run: `npm run dev`

### 4. Vite React
- Directory: `4-vite-react/`
- Setup: `npm create vite@latest` (React template)
- Run: `npm run dev`

### 5. Vite Vue
- Directory: `5-vite-vue/`
- Setup: `npm create vite@latest` (Vue template)
- Run: `npm run dev`

### 6. Create React App
- Directory: `6-create-react-app/`
- Setup: `npx create-react-app`
- Run: `npm start`

### 7. Angular SPA
- Directory: `7-angular-spa/`
- Setup: `ng new` (requires @angular/cli)
- Run: `ng serve`

### 8. Vue SPA (npm init vue)
- Directory: `8-vue-init/`
- Setup: `npm init vue@latest`
- Run: `npm run dev`
